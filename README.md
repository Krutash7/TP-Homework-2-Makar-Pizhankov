## Описание

Проект представляет собой систему управления рецептами на Python.

Реализованы следующие классы:

`Ingredient` — ингредиент с названием, количеством и единицей измерения;
`Recipe` — рецепт блюда с набором ингредиентов;
`DietaryRecipe` — рецепт с указанием диетической категории;
`ShoppingList` — список покупок, автоматически объединяющий одинаковые ингредиенты из разных рецептов.

Для проверки корректности работы реализованы тесты с использованием библиотеки pytest.

Запуск тестов:

```bash

```

Пример создания рецепта:

```python

recipe.add_ingredient(Ingredient("Мука", 500, "г"))
```
